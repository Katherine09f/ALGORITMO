Pregunta de Opción Múltiple

¿Cuál es la diferencia principal entre un algoritmo iterativo y uno recursivo?

  • a. Un algoritmo recursivo es más fácil de entender y programar que uno iterativo.
  • b. Un algoritmo iterativo repite un bloque de código utilizando bucles, mientras que un algoritmo recursivo se llama a sí mismo para resolver subproblemas.
  • c. Un algoritmo iterativo no puede resolver problemas complejos, mientras que uno recursivo sí.
  • d. Un algoritmo iterativo siempre es más eficiente que uno recursivo.
